Actividad en Clase

Para esta actividad, se nos pidi trabajar con uno de los grupos de datos disponibles en el dataset Datasaurus Dozen. Me fue asignado el grupo bullseye. A continuacin, guardo el data frame correspondiente a este grupo y realizo las operaciones indicadas.

Consideraciones sobre un data frame en R:

  • Las columnas deben tener un nombre.
  • Las columnas pueden tener diferentes tipos de datos.
  • Cada columna debe tener el mismo nmero de datos.
# Filtrar y guardar el data frame del grupo "bullseye"
bullseye_data <- datasaurus_dozen %>% 
  filter(dataset == "bullseye")
