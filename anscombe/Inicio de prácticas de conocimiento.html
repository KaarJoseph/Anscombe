3. Visualizaci√≥n de los Datos de Anscombe con Gr√°ficos de Dispersi√≥n¬∂

In¬†[6]:
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns

# Cargar el archivo CSV (ya lo tienes cargado)
archivo_csv = 'df_anscombe.csv'
df = pd.read_csv(archivo_csv)

# Establecer el estilo de Seaborn
sns.set(style="whitegrid")

# Crear la figura y los subplots
fig, axs = plt.subplots(2, 2, figsize=(10, 8))
fig.suptitle("Anscombe's quartet")

# Gr√°fica para el Grupo 1
sns.scatterplot(ax=axs[0, 0], x='x', y='y', data=df[df['group'] == 1])
axs[0, 0].set_title('Group 1')

# Gr√°fica para el Grupo 2
sns.scatterplot(ax=axs[0, 1], x='x', y='y', data=df[df['group'] == 2])
axs[0, 1].set_title('Group 2')

# Gr√°fica para el Grupo 3
sns.scatterplot(ax=axs[1, 0], x='x', y='y', data=df[df['group'] == 3])
axs[1, 0].set_title('Group 3')

# Gr√°fica para el Grupo 4
sns.scatterplot(ax=axs[1, 1], x='x', y='y', data=df[df['group'] == 4])
axs[1, 1].set_title('Group 4')

# Ajustar el layout
plt.tight_layout()
plt.subplots_adjust(top=0.9)
plt.show()

Estos gr√°ficos muestran que cada grupo tiene una relaci√≥n √∫nica entre (x) e (y), revelando patrones de distribuci√≥n, alineaci√≥n lineal y posibles outliers. Este an√°lisis visual refuerza que las estad√≠sticas por s√≠ solas no capturan la naturaleza completa de los datos.

# Establecer el estilo de Seaborn
sns.set(style="whitegrid")

# Crear la figura y los subplots para los boxplots
fig, axs = plt.subplots(2, 2, figsize=(15, 12))
fig.suptitle("Boxplots por Grupo con Outliers")

# Boxplot para el Grupo 1 (ajustando 'whis')
sns.boxplot(ax=axs[0, 0], x='group', y='y', data=df[df['group'] == 1], whis=1.0)
axs[0, 0].set_title('Boxplot - Grupo 1')

# Boxplot para el Grupo 2 (ajustando 'whis')
sns.boxplot(ax=axs[0, 1], x='group', y='y', data=df[df['group'] == 2], whis=1.0)
axs[0, 1].set_title('Boxplot - Grupo 2')

# Boxplot para el Grupo 3 (ajustando 'whis')
sns.boxplot(ax=axs[1, 0], x='group', y='y', data=df[df['group'] == 3], whis=1.0)
axs[1, 0].set_title('Boxplot - Grupo 3')

# Boxplot para el Grupo 4 (ajustando 'whis')
sns.boxplot(ax=axs[1, 1], x='group', y='y', data=df[df['group'] == 4], whis=1.0)
axs[1, 1].set_title('Boxplot - Grupo 4')

# Ajustar el layout
plt.tight_layout()
plt.subplots_adjust(top=0.9)
plt.show()

Los box plots destacan las diferencias en la distribuci√≥n y presencia de outliers en cada grupo, reforzando la idea de que, aunque los grupos compartan estad√≠sticas similares, sus distribuciones tienen variaciones que se pueden observar en esta visualizaci√≥n.
